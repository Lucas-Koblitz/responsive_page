Introduction

HTML is the fundamental language to create the structure and content of web pages. It creates the structure necessary to use other tecnologies, such as CSS and JavaScript, here are some other motives:

  • Whether you want to create a blog, personal website, or contribute to online platforms, understanding HTML is essential.
  • Many jobs related to web development, digital marketing, and content creation, require at least a basic knowledge of HTML. It's a valuable skill that can enhance your employability.

Learning HTML is the first step towards building a strong foundation in web development.

Basic Structure

All html files start with the DOCTYPE Statement:

<!DOCTYPE html>

They should also contain a <html> tag that contains all the code. And also, for readability pourposes, it's better practice if the meta-content, inside <head> and the content is displayed inside <body>


<!DOCTYPE html> 
<html>
  <head> 

  </head> 
  <body> 

  </body> 
</html>

Understanding the basic structure of an HTML document is crucial for creating well-organized and readable web pages.

Simple Statements

You can create simple statements using the following tags, they differentiate themselves in size.

  • Heading 1 <h1>

  • Heading 2 <h2>

  • Heading 3 <h3>

  • Heading 4 <h4>

  • Paragraph <p>

Choosing the appropriate HTML tags for your content helps in creating a well-formatted and visually appealing webpage.

Hypertext Reference

If you want to make your website more interactive, you can start using hyperlinks, to reference to other sites, or to direct you to other parts of the site, like in the example below:

<a href="(link to site)">Text to link</a>

Or if you want to reference at the same page, do the following

<a href="#(id of element)">Text to link</a>

Hyperlinks provide navigation and connectivity, enhancing the user experience on your website.
